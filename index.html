A naive approach


class PostsController < ApplicationController
  def show
    @post = Post.find params[:id]
    render :text => %[{
      title : "#{@post.title}",
      body  : "#{@post.body}"
     }]
  end
end

This is a
losing battle.

Issues with this approach


  • Building strings directly is error prone.
  • The contoller is overloaded.
  • Rails solves this problem already.

The Rails Way


class PostsController < ApplicationController
  def show
    @post = Post.find(params[:id])
  end
end
